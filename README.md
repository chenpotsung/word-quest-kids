# Word Quest Kids 英文尋寶隊

一個為小學三年級設計的英文單字遊戲網站，以叢林尋寶為主題。

## 立即遊玩

https://word-quest-kids.kuascong.chatgpt.site

## 遊戲內容

- 看圖選單字
- 拼字挑戰
- 配對翻牌
- 快問快答
- 星星獎勵與本機進度保存

## 本機執行

需要 Node.js 22.13 或更新版本。

```bash
npm install
npm run dev
```

接著開啟終端機顯示的本機網址。

## 主要技術

React、TypeScript、Vinext、Vite、Tailwind CSS。
